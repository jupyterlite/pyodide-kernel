# Contributing

## Prerequisites

- `python >=3.8`
- `nodejs >=18,<19`

## Quick Start

```bash
git clone https://github.com/jupyterlite/pyodide-kernel
cd pyodide kernel
npm run quickstart
```

```bash
jlpm serve
```
