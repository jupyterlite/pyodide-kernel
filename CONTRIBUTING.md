# Contributing

## Prerequisites

- `python >=3.8`
- `nodejs >=18,<19`

## Quick Start

```bash
git clone https://github.com/jupyterlite/pyodide-kernel
cd pyodide kernel
jlpm bootstrap
```

## Build

```bash
jlpm build
```

## Distributions

```
jlpm dist
```
